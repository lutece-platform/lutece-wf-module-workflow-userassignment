# Plugin userassignment

## Introduction

This module contains workflow tasks for the plugin **plugin-userassignment** .

## Task Type 

 **(Userassignment) Assign a notification** 

 **(Userassignment) Assign a response** : It is possible to configure this task to assign a response to the current user or to an entity of choice.

 **(Userassignment) Assign a response to the current user** : Assign a response to the current user directly without intermediate screens.

 **(Userassignment) Unassign a response** 

 **(Userassignment) Unassign a response automatically**
